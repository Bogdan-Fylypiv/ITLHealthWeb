## üö® Common Issues & Solutions

### Issue 1: CSS Not Loading
**Symptom:** Page has no styling  
**Solution:**
- Check CSS file path in Orders.aspx
- Verify Orders.css is in Content folder
- Clear browser cache (Ctrl+F5)
- Check IIS MIME types for .css files

### Issue 2: Page Not Found (404)
**Symptom:** Cannot navigate to Orders.aspx  
**Solution:**
- Verify file is in Pages folder
- Check file name spelling
- Rebuild solution
- Check IIS application pool is running

### Issue 3: Designer File Errors
**Symptom:** Red squiggles under control names  
**Solution:**
- Rebuild solution
- Close and reopen Visual Studio
- Right-click Orders.aspx ‚Üí "Convert to Web Application"

### Issue 4: Master Page Not Applied
**Symptom:** Page doesn't have header/footer  
**Solution:**
- Check MasterPageFile attribute in Orders.aspx
- Verify Site.Master exists
- Check ContentPlaceHolderID matches
